## Описание <a name="описание" id="описание"></a>
Проект **TARO** — это сервис для гадания на картах Таро. 
Вы можете получить одну из 78 карт с её прямым и перевёрнутым значением. (В будущем добавятся другие расклады.)

## Требования  <a name="требования" id="требования"></a>
- **Java 25**, **Gradle 9.2**, **PostgreSQL 18** (для запуска через Gradle)
- **Docker** (для запуска через Docker Compose)

## Инструкция по запуску <a name="запуск" id="запуск"></a>
### Подготовка окружения
В файле `.env` хранятся настройки.

### Запуск через Docker (с поддержкой HTTPS) <a name="запуск_docker" id="запуск_docker"></a>  
В корне проекта выполнить команду:
``` bash
docker-compose up --build
```
* **HTTPS**: Приложение доступно по адресу `https://localhost` (или по IP из .env).
* **SSL**: При первом входе браузер выдаст предупреждение о самоподписанном сертификате. Нажмите "Дополнительно" -> "Перейти на сайт (небезопасно)".
* **База данных**: Поднимается автоматически на порту 5432 (PostgreSQL 18).

### Запуск через Gradle <a name="запуск_gradle" id="запуск_gradle"></a>  
Перед запуском команды.
Требуется установленный и запущенный PostgreSQL, создать БД tarodb, настройки профиля и подключения в `application.properties` 
``` bash
./gradlew bootRun --args='--spring.profiles.active=dev'
```
